Machine Learning

BukuKu

  • Role: Machine Learning Model Developer and UI/UX Designer
  • Software: Scikit-Learn (surprise), TensorFlow, Keras, Pandas, Numpy, BeautifulSoup
  • Category: E-Book Mobile Application
  • Project date: 05/2023 - 06/2023
  • Kaggle Dataset
  • Github Repo
  • Figma Page

BukuKu is an innovative application that is used for people who want to find and read books according to their interests. The aim of this app is to provide relevant and personalized book recommendations based on the user's interests and needs.

  • Responsible to build Collaborative-filtering, designing the application interface using Figma, and coordinate with other division to deploy Machine Learning model to Google Cloud Platform.
  • Research appropriate dataset for the model.
  • Pre-process the dataset to extract available book data by scraping it from goodreads.com, using BeautifulSoup and then further pre-processing it using Microsoft Excel. Data reduced from 982.000 rows to 609.206 rows.
  • Built the model with Scikit-Learn and TensorFlow, using Probabilistic Matrix Factorization method, resulting in 0.9 RMSE.
  • Predicted result of recommended books are combined with result from Content-based filtering model, which results in hybrid recommendation.
